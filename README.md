# 🎲 Number Guessing Game

A fun and interactive number guessing game built using **HTML**, **CSS**, and **JavaScript**.

The player has to guess a number between 1 and 100. The app gives hints like **"Too High"** or **"Too Low"** until the correct number is guessed. It also tracks the number of attempts and includes a **Restart** button.

---

## 🚀 Features

- 🎯 Random number from 1 to 100
- 💡 Hints for each guess (too high / too low)
- 🔄 Restart game option
- 🧠 Logic handled with JavaScript
- 🧼 Clean and responsive UI

---

## 💻 Technologies Used

- HTML
- CSS
- JavaScript
